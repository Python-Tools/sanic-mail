Sanic-mail.

class sanic_mail.Sanic_Mail(app=None) → None[source]¶

Bases: object

sanic的邮件发送插件.

使用后发送邮件会被绑定在`app对象`上,支持协程`send_email`, 也支持方法`send_email_nowait`,其中`send_email_nowait`意为将任务交给协程发送而不等待发送完毕, 会返回发送的task.
